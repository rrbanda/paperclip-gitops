# Red Hat Americas AI Platform Team

This company manages the AI agent workforce supporting Red Hat's Americas AI Platform sales organization.

## Organization

- **AI Sales Operations Lead** - orchestrates strategic requests and coordinates specialist agents
- **RHOAI Solutions Lead** - coordinates technical specialist agents for SSA support
- **17 Chief of Staff agents** - personal assistants for each team member
- **13 Specialist agents** - domain-specific experts (pitch building, RFQs, demos, etc.)
- **5 Mortgage AI agents** - connected to external mortgage-ai system via HTTP adapter

## Adapter Types

- `gemini_local` - Gemini CLI for autonomous task execution
- `http` - HTTP adapter for external API integrations (mortgage-ai system)
